# Paul Graham Notes

This is the compact source note behind the skill.

## Main takeaways

- Generate many credible options before choosing.
- A usable name beats a long hunt for a perfect one.
- The domain path is part of name quality.
- Good `.com` names still exist if you search broadly enough.
- Skip mediocre names that are already parked or behind "make offer".
- Pay real money only for names that are genuinely strong enough to justify it.
